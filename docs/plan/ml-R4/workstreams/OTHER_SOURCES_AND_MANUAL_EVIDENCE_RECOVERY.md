# Other Sources and Manual Evidence Recovery Workstream

## Scope

Recover evidence for SolidiFI, SmartBugs Curated, Web3Bugs, DISL, DeFiHackLabs, and any source present in the actual active export.

## Per-source fields

- source pin/acquisition;
- unit of analysis;
- positive mechanism;
- negative/absence semantics;
- manual verification;
- injection/exploit evidence;
- class coverage;
- compiler-era coverage;
- duplicate/project structure;
- known selection bias;
- crosswalk and tier declarations;
- prior audits;
- intended role.

## Manual evidence

Search for contract reviews outside source-specific directories, including issue notes, CSVs, benchmark cases, AI outputs, local reports, and commit-attached artifacts.

## Output

Populate previous-evidence and artifact registers; propose only specific gaps.
