## Retrain Gate

```
DO NOT RETRAIN without running validate_graph_dataset.py first
  — validates edge_attr presence, shape [E], and value range [0, 6] (disk types 0–6)
  — imports NUM_EDGE_TYPES from graph_schema.py

Retrain evaluation protocol (v5.2):
  deduped split:  ml/data/splits/deduped/ — train=31,092 / val=6,661 / test=6,667 (FROZEN)
  eval_threshold: 0.35 during training-time patience tracking (NOT inference threshold)
                  minority classes cluster at 0.35–0.50; using 0.50 causes ±0.04 F1 noise
  success:        val F1-macro (at threshold=0.35) improves epoch-over-epoch, then
                  tuned per-class thresholds must exceed v4 floors on test split
  floor:          no single class drops > 0.05 F1 below v4 tuned values (listed below)
  rollback:       if tuned F1 < v4 fallback after completion: revert to v4 checkpoint
  MLflow:         experiment "sentinel-v5.2"; compare against "sentinel-retrain-v4"

v4 per-class F1 floors (v5.2 per-class tuned F1 must exceed floor = v4_F1 − 0.05):
```
