## Temperature Scaling

Post-training calibration that learns a single temperature parameter per class to improve probability calibration without changing predictions.

**Usage:**
```bash
python ml/scripts/calibrate_temperature.py \
    --checkpoint ml/checkpoints/<run>_best.pt \
    --output ml/calibration/
```

**Output:**
- `temperatures_<run>.json` — `{class_name: temperature_value}`
- `temperatures_<run>_stats.json` — ECE before/after, per-class calibration metrics
- `temperatures_<run>_ece_comparison.png` — reliability diagram

**Current state:** Run 12 calibration complete. Active checkpoint uses Run 12 temperatures.
