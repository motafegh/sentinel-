# SENTINEL Contracts

Foundry project for all on-chain SENTINEL components.

## Directory Structure

```
contracts/
+-- foundry.toml              # Foundry config (solc 0.8.20 default)
+-- src/
|   +-- AuditRegistry.sol     # UUPS upgradeable audit registry
|   +-- IZKMLVerifier.sol     # Interface - ABI bridge to EZKL verifier
|   +-- SentinelToken.sol     # ERC-20 staking token (SNTL)
|   +-- ZKMLVerifier.sol      # <- EZKL-generated, NOT in git (see below)
+-- test/                     # Foundry tests (forge test)
+-- script/                   # Deployment scripts (forge script)
+-- out/                      # Build artifacts (gitignored)
    +-- AuditRegistry.sol/
        +-- AuditRegistry.json  <- audit_server.py reads this ABI
```

## Build

```bash
# One-time: install OZ dependencies
cd contracts
forge install OpenZeppelin/openzeppelin-contracts
forge install OpenZeppelin/openzeppelin-contracts-upgradeable

# Build AuditRegistry + SentinelToken (solc 0.8.20)
forge build

# ABI consumed by audit_server.py will be at:
# contracts/out/AuditRegistry.sol/AuditRegistry.json
```

## ZKMLVerifier - Special Handling

`ZKMLVerifier.sol` is auto-generated by EZKL from `zkml/ezkl/verification_key.vk`.
It uses Halo2 assembly with deprecated opcodes - requires `solc 0.8.17`.

```bash
# 1. Generate verifier (zkml module, step 4 of EZKL pipeline)
cd zkml
poetry run python -m src.ezkl.setup_circuit

# 2. Copy output into contracts/src/
cp zkml/ezkl/ZKMLVerifier.sol contracts/src/ZKMLVerifier.sol

# 3. Compile with correct solc
solc-select use 0.8.17
cd contracts && forge build --contracts src/ZKMLVerifier.sol

# 4. Restore default
solc-select use 0.8.20
```

Must redeploy ZKMLVerifier whenever CrossAttentionFusion output_dim changes.
Currently locked at 128 - SENTINEL-SPEC s6 Critical Constraints.

## Deployment (Sepolia)

```bash
export SEPOLIA_RPC_URL=<your-rpc>
export DEPLOYER_PRIVATE_KEY=<your-key>
export ETHERSCAN_API_KEY=<your-key>

forge script script/Deploy.s.sol \
  --rpc-url $SEPOLIA_RPC_URL \
  --broadcast \
  --verify
```

## Critical Constraints (SENTINEL-SPEC s6)

| Constraint | Value |
|---|---|
| solc - all contracts except ZKMLVerifier | 0.8.20 |
| solc - ZKMLVerifier.sol only | 0.8.17 |
| publicSignals[64] | Score field element index in EZKL proof |
| Scale factor | 8192 (2^13, from settings.json) |
| Little-endian Python decode | int.from_bytes(bytes.fromhex(x), byteorder='little') |
| ABI path for audit_server.py | contracts/out/AuditRegistry.sol/AuditRegistry.json |
